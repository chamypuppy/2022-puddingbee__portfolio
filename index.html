벚꽃 푸딩톡

개발 언어 기획 의도 주요 기능 코딩 효과 코드 링크
HTML   CSS   Javascript

노마드코더 코코아톡 클론 코딩 실습 중
HTML과 CSS의 기초를 얼마만큼 이해하고 있는지 확인해보고 직접 연습해보기 위해
실제 카카오톡 디자인을 기반으로 코딩하게 되었습니다.
또한 Javascript와 input을 자유자재로 넣어보고
preventDefault() 함수를 통해 submit에 제한을 걸어보는
연습을 하는 데에 집중하였습니다.

  • 벚꽃 푸딩톡 로그인 페이지 구현(form과 input 태그)
  • 하단 네비게이션 탭 구현
  • 친구 페이지 구현
  • 채팅창 구현 (flex 박스로 구현 - 너비에 따른 레이아웃의 무너짐을 고려)
  • 뷰 페이지 구현
  • 쇼핑 페이지 구현(flex 박스 구현)
  • 더보기 페이지 구현(카테고리 flex 박스로 정렬)

HTML

  • HTML에 다양한 태그가 있고 CSS와 Javascript를 사용하기 이전에 HTML의 뼈대를 탄탄하게 구성하는 것이 가장 중요하다는 것을 알게 되었습니다.

CSS

  • HTML문서에서 link를 걸어 CSS문서를 하나 하나 연결하는 방법 외에도 모든 CSS를 총괄하는 style.css 문서 파일을 생성하여 세부 CSS 파일을 @import해서 관리하는 효율적인 방법이 얼마나 유용한지에 대해 배울 수 있었습니다.
  • flex 박스를 통해 브라우저 너비의 가변성에 대응하는 유연한 레이아웃을 구현할 수 있었습니다.

Javascript

  • Javascript의 function과 preventDefault() 그리고 toggle()을 통해 사용자의 클릭이나 입력에 반응하는 세밀한 사용자의 상호작용 요소를 구현할 수 있었습니다.
